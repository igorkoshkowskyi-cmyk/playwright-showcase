# Playwright Showcase

E2E test automation showcase project demonstrating Playwright + TypeScript skills.

## Stack

- Playwright + TypeScript
- Page Object Model
- Fixtures + storage state
- Sharded CI on GitHub Actions

## What's covered

- Authentication via shared storage state
- POM with thin BasePage contract
- Component objects (SideMenu)
- ...

## Run locally

\`\`\`bash
npm install
npx playwright install
npx playwright test
\`\`\`
